Google Calendar project

This Python script automates the process of transferring assignments and tasks from Canvas to Google Calendar or Google Tasks. It connects to Canvas using its API to fetch assignment due dates, and then uses Google Calendar and Google Tasks APIs to create corresponding events or tasks. This integration helps keep track of upcoming deadlines in a more centralized and organized manner, ensuring that nothing is missed.
